## ⚠️ 부족한 핵심 시스템

### 1. **NFT 속성/레어리티 시스템** ⭐⭐⭐ (가장 중요)

**현재 상태**:
- NFT에 `stars: 2-4개`만 있음
- 레어리티 등급 없음
- 속성 시스템 없음

**필요한 것**:
- **레어리티 등급**: Common (70%), Rare (20%), Epic (8%), Legendary (2%)
- **레어리티별 시각적 차별화**: 색상, 배지, 테두리
- **속성 시스템**:
  - `miningEfficiency`: 채굴 효율 증가 (예: Common +10%, Rare +25%, Epic +50%, Legendary +100%)
  - `huntingDamage`: 사냥 데미지 증가
  - `explorationReward`: 탐험 보상 증가
  - `productionSpeed`: 생산 속도 증가
- **레어리티별 스탯 범위**: 레어할수록 더 높은 스탯

**영향**: 
- 게임플레이 깊이 증가
- NFT 가치 차별화
- 블록체인 민팅 시 메타데이터로 사용 가능

---

### 2. **NFT를 게임에서 실제 사용** ⭐⭐⭐ (가장 중요)

**현재 상태**:
- 슬롯에 NFT 배치만 가능
- 실제 게임 효과 없음 (보상 증가 없음)

**필요한 것**:
- MainScreen의 각 모드에서 슬롯 NFT 확인
- NFT 속성에 따른 보상 배수 적용
- 예시:
  - 채굴: `baseReward * (1 + miningEfficiency 합계)`
  - 사냥: `baseDamage * (1 + huntingDamage 합계)`
  - 탐험: `baseReward * (1 + explorationReward 합계)`
  - 생산: `baseSpeed * (1 + productionSpeed 합계)`

**구현 위치**:
- `MainScreen.vue`의 각 모드 클릭 핸들러
- 슬롯에 배치된 NFT의 속성 합산
- 보상 계산 시 배수 적용

**영향**:
- NFT의 실제 가치 실현
- 게임플레이 동기 부여
- NFT 수집의 의미 부여

---

### 3. **NFT 레벨업 시스템** ⭐⭐

**현재 상태**:
- NFT에 `level: 1`만 있고 레벨업 불가

**필요한 것**:
- 게임 플레이 시 슬롯 NFT에 경험치 부여
- 경험치가 쌓이면 레벨업 가능
- 레벨업 시 스탯 증가
- 레벨업 비용 (코인 소모, 레벨이 높을수록 비용 증가)

**구현 방법**:
- 각 모드 클릭 시 슬롯 NFT에 경험치 부여
- 인벤토리에서 NFT 상세 모달에 "레벨업" 버튼 추가
- 레벨업 시 스탯 증가 애니메이션

**영향**:
- 장기적 성장 요소
- NFT 가치 증가
- 게임플레이 지속성

---

### 4. **블록체인 NFT 민팅** ⭐⭐⭐ (블록체인 연동 필수)

**현재 상태**:
- FactoryPage에서 고양이 제작만 가능
- 블록체인에 민팅 안됨 (로컬 데이터만)

**필요한 것**:
- ERC-721 또는 ERC-1155 스마트 컨트랙트
- 실제 블록체인에 NFT 민팅
- 민팅 비용 (가스비 + 민팅 수수료)
- 트랜잭션 진행 상황 표시
- 민팅 후 블록체인에서 소유권 확인

**구현 방법**:
- FactoryPage에 "블록체인에 민팅" 버튼 추가
- 스마트 컨트랙트 ABI 정의
- `ethers.js`로 컨트랙트 호출
- 민팅 성공 후 NFT에 `tokenId`, `contractAddress` 추가

**영향**:
- 진정한 NFT 게임이 됨
- 소유권 보장
- 거래 가능

---

### 5. **NFT 마켓플레이스** ⭐⭐⭐ (경제 순환 필수)

**현재 상태**:
- NFTPage에 더미 데이터만 있음

**필요한 것**:
- 내 NFT 판매 기능 (ETH/토큰으로)
- 다른 사용자 NFT 구매 기능
- 가격 설정
- 거래 내역
- 필터링 (레어리티, 가격대, 속성)

**구현 방법**:
- 스마트 컨트랙트 기반 마켓플레이스
- 또는 중앙화 마켓플레이스 (localStorage 기반, 나중에 블록체인 연동)

**영향**:
- 경제 순환
- NFT 가치 실현
- 게임 지속성

---

### 6. **NFT 조합/업그레이드 시스템** ⭐⭐

**현재 상태**:
- NFT 제작만 있고 조합 없음

**필요한 것**:
- 같은 종류의 NFT 2-3개를 조합하여 더 강한 NFT 제작
- 레어리티 업그레이드 (Common 3개 → Rare 1개)
- 조합 비용 (코인/고양이 파편)
- 조합 성공률 (100% 또는 확률)

**구현 방법**:
- FactoryPage에 "조합" 탭 추가
- 조합할 NFT 선택 UI
- 조합 결과 계산 및 생성

**영향**:
- 게임플레이 다양성
- NFT 소각 메커니즘
- 희소성 유지

---

### 7. **NFT 스테이킹 시스템** ⭐⭐

**필요한 것**:
- NFT를 스테이킹하여 수동 수익 획득
- 스테이킹 기간에 따른 보상
- 여러 NFT 동시 스테이킹
- 언스테이킹 시 보상 지급

**게임플레이**:
- 게임을 안 해도 NFT가 자동으로 수익 생성
- 스테이킹한 NFT는 게임에서 사용 불가 (트레이드오프)

**영향**:
- 오프라인 수익
- NFT 가치 증가
- 게임 지속성

## 🎮 현재 게임플레이 루프

```
클릭 → 포인트 획득 → 코인 획득 → 고양이 파편 획득
  ↓
FactoryPage에서 고양이 제작
  ↓

  ↓
슬롯에 배치 (하지만 효과 없음)

게임 플레이 (NFT 효과 없이 진행)
```

```

```
