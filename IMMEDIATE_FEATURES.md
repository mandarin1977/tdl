# 지금 바로 진행 가능한 기능들

## ✅ 스마트 컨트랙트 없이도 가능한 기능들

### 1. **NFT 속성/레어리티 시스템** ⭐⭐⭐ (가장 중요)
**현재 상태**: 
- FactoryPage에서 고양이 제작 시 `stars: 2-4개`만 있음
- InventoryPage에 스탯은 있지만 레어리티 시스템 없음

**바로 할 수 있는 것**:
- 레어리티 등급 추가 (Common, Rare, Epic, Legendary)
- 레어리티별 색상/배지 표시
- 레어리티에 따른 속성 범위 설정
- 레어리티별 드롭 확률 설정

**구현 방법**:
```javascript
// FactoryPage에서 제작 시
const rarity = calculateRarity() // 확률 기반
// Common: 70%, Rare: 20%, Epic: 8%, Legendary: 2%

const newCat = {
  id: Date.now(),
  imageId: newCatImageId.value,
  name: `Cat ${newCatImageId.value}`,
  rarity: rarity, // 'common' | 'rare' | 'epic' | 'legendary'
  stars: getStarsByRarity(rarity), // 레어리티에 따라 별 개수
  level: 1,
  stats: generateStatsByRarity(rarity), // 레어리티에 따라 스탯 범위
  // 게임 속성
  miningEfficiency: getEfficiencyByRarity(rarity),
  huntingDamage: getDamageByRarity(rarity),
  explorationReward: getRewardByRarity(rarity),
  productionSpeed: getSpeedByRarity(rarity)
}
```

**영향**:
- 게임플레이 깊이 증가
- NFT 가치 차별화
- 나중에 블록체인 민팅 시 메타데이터로 사용 가능

---

### 2. **NFT 레벨업 시스템** ⭐⭐
**현재 상태**: 
- 고양이에 `level: 1`만 있고 레벨업 기능 없음

**바로 할 수 있는 것**:
- 게임 플레이로 NFT 경험치 획득
- 경험치가 쌓이면 레벨업
- 레벨업 시 스탯 증가
- 레벨업 비용 (코인 소모)

**구현 방법**:
- 채굴/사냥/탐험/생산 시 슬롯에 배치된 NFT에 경험치 부여
- 레벨업 버튼 추가 (인벤토리에서)
- 레벨업 시 스탯 증가 애니메이션

---

### 3. **NFT 조합 시스템** ⭐⭐
**현재 상태**: 
- 고양이 제작만 있고 조합 기능 없음

**바로 할 수 있는 것**:
- 같은 종류의 NFT 2-3개를 조합하여 더 강한 NFT 제작
- 레어리티 업그레이드 (Common 3개 → Rare 1개)
- 조합 비용 (코인/고양이 파편)
- 조합 성공률 (100% 또는 확률)

**구현 방법**:
- FactoryPage에 "조합" 탭 추가
- 조합할 NFT 선택 UI
- 조합 결과 계산 및 생성

---

### 4. **인벤토리 개선** ⭐⭐
**현재 상태**: 
- InventoryPage에 더미 데이터 사용
- 실제 게임 데이터와 연결 안됨
- 필터링/정렬 기능 없음

**바로 할 수 있는 것**:
- 실제 게임 인벤토리 데이터 표시
- 레어리티별 필터링
- 레벨별 정렬
- 검색 기능
- NFT 상세 정보 모달
- NFT 선택/해제 기능

---

### 5. **NFT를 게임에서 실제 사용** ⭐⭐⭐
**현재 상태**: 
- 슬롯에 배치만 되고 실제 효과 적용 안됨

**바로 할 수 있는 것**:
- NFT 속성에 따른 게임 효율 증가
- 레어리티에 따른 보상 증가
- 레벨에 따른 추가 보너스
- NFT 조합 효과 (같은 종류 배치 시 시너지)

**구현 방법**:
- MainScreen의 각 모드에서 슬롯 NFT 확인
- NFT 속성에 따라 보상 배수 적용
- 예: Legendary NFT = 5배 보상, Rare = 2배 보상
