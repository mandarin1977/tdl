# NFT 클리커 게임 기능 제안서

## 📊 현재 상태 분석

### ✅ 구현된 기능
- 클리커 게임 메커니즘 (채굴, 사냥, 탐험, 생산)
- 고양이 NFT 시스템 (30종류)
- 지갑 연결 기능 (MetaMask)
- 게임 리소스 관리 (포인트, 코인, 고양이 파편)
- 레벨 시스템

### ⚠️ 부족한 부분
- 실제 블록체인 NFT 민팅
- NFT 마켓플레이스
- NFT 스테이킹/보상 시스템
- NFT 속성/레어리티 시스템

---

## 🎯 필수 기능 (Must Have)

### 1. **블록체인 NFT 민팅 시스템** ⭐⭐⭐
**현재**: FactoryPage에서 고양이 제작은 하지만 블록체인에 민팅 안됨

**필요한 것**:
- ERC-721 또는 ERC-1155 스마트 컨트랙트
- 실제 블록체인에 NFT 민팅
- 민팅 비용 (가스비 + 민팅 수수료)
- 민팅 후 블록체인에서 소유권 확인

**구현 포인트**:
- FactoryPage에서 "NFT로 민팅" 버튼 추가
- 트랜잭션 진행 상황 표시
- 민팅 성공 후 NFT 목록에 추가

---

### 2. **NFT 속성/레어리티 시스템** ⭐⭐⭐
**현재**: 고양이에 별(2-4개)만 있음

**필요한 것**:
- **레어리티 등급**: Common, Rare, Epic, Legendary
- **속성 시스템**: 
  - 채굴 효율 증가
  - 사냥 데미지 증가
  - 탐험 보상 증가
  - 생산 속도 증가
- **레벨 시스템**: NFT 레벨업 (경험치 획득)
- **스탯 시스템**: 공격력, 방어력, 효율 등

**게임플레이 영향**:
- 레어한 NFT일수록 게임 효율 증가
- NFT 레벨업으로 더 강해짐
- NFT 조합으로 더 강한 NFT 제작

---

### 3. **NFT 마켓플레이스** ⭐⭐⭐
**현재**: NFTPage에 더미 데이터만 있음

**필요한 것**:
- **판매 기능**: 내 NFT를 ETH/토큰으로 판매
- **구매 기능**: 다른 사용자의 NFT 구매
- **가격 설정**: 판매 가격 설정
- **거래 내역**: 판매/구매 히스토리
- **필터링**: 레어리티, 가격대, 속성별 필터

**구현 방식**:
- 스마트 컨트랙트 기반 마켓플레이스
- 또는 중앙화 마켓플레이스 (OpenSea 연동)

---

### 4. **NFT 스테이킹 시스템** ⭐⭐
**필요한 것**:
- NFT를 스테이킹하여 수동 수익 획득
- 스테이킹 기간에 따른 보상
- 여러 NFT 동시 스테이킹
- 언스테이킹 시 보상 지급

**게임플레이**:
- 게임을 안 해도 NFT가 자동으로 수익 생성
- 스테이킹한 NFT는 게임에서 사용 불가 (트레이드오프)

---

## 🚀 고급 기능 (Nice to Have)

### 5. **NFT 조합/업그레이드 시스템** ⭐⭐
**필요한 것**:
- 같은 종류의 NFT 2-3개를 조합하여 더 강한 NFT 제작
- 레어리티 업그레이드
- 속성 강화
- 조합 비용 (코인/ETH)

**예시**:
- Common 고양이 3개 → Rare 고양이 1개
- Rare 고양이 2개 → Epic 고양이 1개

---

### 6. **리더보드/랭킹 시스템** ⭐⭐
**필요한 것**:
- 전체 사용자 랭킹
- NFT 소유 개수 랭킹
- 게임 레벨 랭킹
- 주간/월간 리더보드
- 상위 랭커 보상

---

### 7. **퀘스트/미션 시스템** ⭐⭐
**현재**: QuestPage는 있지만 내용 확인 필요

**필요한 것**:
- 일일 퀘스트
- 주간 미션
- 특별 이벤트 퀘스트
- 퀘스트 완료 보상 (코인, NFT, 경험치)

**예시**:
- "10번 채굴하기" → 보상: 100 코인
- "레어 NFT 1개 소유하기" → 보상: 특별 NFT
- "일주일 연속 출석" → 보상: 레전더리 NFT

---

### 8. **소셜 기능** ⭐
**필요한 것**:
- 친구 시스템
- NFT 선물하기
- 길드/클랜 시스템
- 채팅 기능

---

### 9. **이벤트/시즌 시스템** ⭐⭐
**필요한 것**:
- 시즌별 한정 NFT
- 이벤트 기간 보상 증가
- 특별 이벤트 NFT 민팅
- 시즌 패스 (유료)

**예시**:
- "여름 시즌" → 해변 고양이 NFT
- "할로윈 이벤트" → 호박 고양이 NFT
- "크리스마스 이벤트" → 산타 고양이 NFT

---

### 10. **토큰 이코노미** ⭐⭐⭐
**필요한 것**:
- **게임 토큰 발행** (ERC-20)
- 토큰으로 NFT 구매
- 게임 플레이로 토큰 획득
- 토큰 스테이킹
- 토큰 소각 메커니즘

**경제 모델**:
- 게임 플레이 → 토큰 획득
- 토큰 → NFT 구매/업그레이드
- NFT 판매 → 토큰/ETH 획득
- 토큰 소각 → 희소성 유지

## 🎮 게임플레이 루프 제안

### 현재 루프
```
클릭 → 포인트 획득 → 고양이 제작 → 게임 효율 증가 → 더 많은 포인트
```

### 개선된 루프
```
클릭 → 포인트/토큰 획득 
  ↓
고양이 파편 수집 
  ↓
NFT 민팅 (블록체인) 
  ↓
NFT 스테이킹 (수동 수익) 또는 게임에서 사용 (활동 수익)
  ↓
더 강한 NFT 조합/업그레이드
  ↓
마켓플레이스에서 판매 → ETH/토큰 획득
  ↓
더 좋은 NFT 구매 또는 게임 리소스 구매
```

- 자동 클릭 (월 구독)

### 4. **시즌 패스**
- 유료 시즌 패스
- 한정 NFT, 특별 보상

## 🔧 기술적 고려사항

### 스마트 컨트랙트
- **ERC-721**: 각 NFT가 고유 (고양이)
- **ERC-1155**: 여러 개 동시 민팅 가능
- **가스비 최적화**: Polygon, Arbitrum 등 L2 체인 고려

### 메타데이터 저장
- **IPFS**: 분산 저장
- **Arweave**: 영구 저장
- **중앙 서버**: 빠른 접근 (하이브리드)

### 백엔드 필요성
- NFT 메타데이터 서버
- 게임 데이터 동기화
- 마켓플레이스 데이터
- 리더보드 데이터

## 🎯 결론

현재 게임은 **기본 클리커 메커니즘**은 잘 갖춰져 있지만, **NFT 게임으로서의 핵심 기능**이 부족합니다.

**가장 우선적으로 필요한 것**:
1. **블록체인 NFT 민팅** (가장 중요!)
2. **NFT 속성 시스템** (게임플레이 깊이)
3. **NFT 마켓플레이스** (경제 순환)

이 세 가지가 있으면 진정한 NFT 게임이 됩니다! 🚀
